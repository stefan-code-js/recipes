# Set Custards without Egg (Carrageenan)

## Technique

Use 0.6–0.9% kappa carrageenan + 0.1–0.2% iota carrageenan for elastic custards.

## Parameters

- **Kappa Carrageenan %**: 0.6–0.9%

- **Iota Carrageenan %**: 0.1–0.2%

- **Service Temp**: 45–55 °C (for aroma bloom)

## Sensory Cues

- **Result**: Elastic, set custard.
