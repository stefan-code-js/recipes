# Vinegar: Thesis & Baseline

**Thesis**: Convert alcohol to \~5% acetic acid using Acetic Acid Bacteria (AAB); oxygen is the throttle.

**Baseline Playbook**:

1. Start with an alcohol base (\~8–10% ABV target).

2. Heat base to **\~70 °C** to knock back yeast.

3. Aerate with a pump/stone; add mother or active vinegar.

4. Track acidity; rack off when target is reached.

**Safety Rail**: Target **\~5% acetic acid**; keep aerated; pasteurize alcohol base at \~70 °C to stop yeasts.
