# Sous-Vide Cooking Techniques

## Temperature Ranges

- **Tender Meat/Poultry**: 60–65 °C for 30–90 min.

- **Tough Meat**: 65–70 °C for 12–48 h.

- **Fish**: 50–55 °C for 10–15 min.

## Storage and Reheating (HACCP)

- **Chilling**: Chill to **≤ 3 °C in ≤ 90 min** if storing.

- **Storage**: Store **≤ 3 days at ≤ 3 °C**.

- **Reheat**: Reheat in bath to service temp.
